🔰 IR System Overview

IR (Infrared) system is used to control TV, AC, Fan, Set-top box devices. When added to VisionGuard: 📱 App can simulate IR remote. 📡 Arduino sends IR signals to control devices. 👉 Basically: Phone = Universal Remote

🧩 Required Components

🔴 IR LED Transmitter Module
📡 IR Receiver Module VS1838B
🎮 ESP32 Dev Board / Arduino
🔧 Resistor + Breadboard + Wires

🔍 Step 1 – Capture IR Codes

#include <IRremote.h>
int recvPin = 15;
IRrecv irrecv(recvPin);
decode_results results;
void setup() { Serial.begin(9600); irrecv.enableIRIn(); }
void loop() { if (irrecv.decode(&results)) { Serial.println(results.value, HEX); irrecv.resume(); } }

👉 Serial Monitor displays codes. Example: Power → 0xFFA25D

📤 Step 2 – Send IR Signals

#include <IRremote.h>
IRsend irsend;
void loop() { irsend.sendNEC(0xFFA25D, 32); delay(5000); }

👉 This turns device ON/OFF

1. 🔰 Arduino System Overview

2. 🧠 Main Components

2.1 🔷 Arduino Board

Use වෙන්න පුළුවන්: Arduino Uno R3, ESP8266 NodeMCU, ESP32 Dev Board 👉 Best option: ESP32 (speed + connectivity)

2.2 📡 Communication Modules

🔹 WiFi (Recommended) ESP8266/ESP32 – Real-time control
🔹 Bluetooth HC-05 – Short distance control
🔹 GSM (Advanced) – SIM-based control (SMS/Call)

2.3 🔌 Output Devices (Controlled Devices)

💡 Lights, 🔊 Speakers/Alarm, 🚪 Gate motor, 🌀 Fans (via 5V Relay Module 4 Channel)

2.4 📊 Input Sensors

🌡️ Temperature – DHT11
🚶 Motion – PIR HC-SR501
🔥 Gas/Smoke – MQ sensors
🚪 Door status – Magnetic sensor

3. 🔁 System Architecture

Android App → ↓ (WiFi/Bluetooth) → Communication Module → ↓ → Arduino/ESP Board → ↓ → Relays & Sensors → ↓ → Devices + Data Feedback

👉 Two-way system: Control (App → Arduino) + Monitoring (Arduino → App)

4. 🎛️ Core Functional Modules

4.1 🔘 Device Control Module

Command example: LIGHT_ON, LIGHT_OFF, FAN_ON → Arduino reads → digitalWrite HIGH/LOW

if(command == "LIGHT_ON"){ digitalWrite(relayPin, HIGH); }

4.2 📡 Communication Handling Module

if (client.available()) { String request = client.readStringUntil('\r'); }

4.3 📊 Sensor Monitoring Module

int motion = digitalRead(pirPin); if(motion == HIGH){ /* alert */ }

4.4 🚨 Alert System

Trigger: Motion/Gas/Door opened → Buzzer ON, App notification, Auto recording

4.5 🔐 Security Logic

Only authenticated commands, OTP login, Optional command encryption

5. 🔄 Automation Features

⏰ Scheduled Tasks: Light ON at 6PM, Fan OFF at 10PM

🤖 Smart Automation: Motion → Light ON, No motion → Light OFF
